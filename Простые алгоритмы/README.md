# Задача 1 (кодирование строки)

В строке буквы с 'a' по 'i' кодируются числами от 1 до 9 соответственно, а буквы с 'j' по 'z' - символами 10# до 26# соответственно. Нужно раскодировать строку

### **Формат ввода**

В первой строке передается строка s (длина строки 1 <= |s| <= 100), которая состоит из цифр и символа '#'. Гарантируется, что данная строка была получена путем кодирования некоторой другой строки

### **Формат вывода**

Нужно вывести раскодированную строку

### Пример
```
123    ->    abc
8512#12#15#    ->    hello
```

## Объяснение решения

Буква у нас может быть закодирована только одним из двух случаев: либо одной цифрой, либо двумя цифрами и символом '#'. На каждой итерации цикла будем проверять i+2 элемент: если он равен '#', то это означает, что перед нами буква среди 'j' - 'z', иначе 'a' - 'i'. Будем пользоваться функциями *ord()* и *chr()*: *ord()* возвращает номер символа в таблице Unicode; *chr()* возвращает символ по коду. Данное решение работает за O(n)

```python
s = input()
i = 0
while (i < len(s)):
    if (i + 2 < len(s) and s[i + 2] == '#'): #проверяем i+2 символ, и смотрим на границы строки
        dig = (ord(s[i]) - ord('0')) * 10 + ord(s[i + 1]) - ord('0') 
        print(chr(ord('a') + dig - 1), end = '')
        i += 3
    else:
        print(chr(ord('a') + ord(s[i]) - ord('0') - 1), end = '')
        i += 1
```

# Задача 2 (Покраска деревьев)

Вася и Маша решили покрасить деревья. Деревья растут вдоль одной прямой и расположены на расстоянии 1 метр друг от друга. Каждое дерево имеет координату (например: 4; -6; 0). Вася и Маша стоят у деревьев с номерами p и q соответственно. Около этих деревьев стоит банка с краской, которую ребята перемащать не могут: они макают в банку кисть и идут красить деревья. Вася может отойти от банки не более чем на v метров, после чего краска на кисти засыхает, Маша - не более чем на m метров. Какое максимальное количество деревьев ребята могут покрасить?

### **Формат ввода**

В первой строке поступают данные для Васи: номер дерева p и количество метров q, которое он может пройти до высыхания краски

Во второй строке такие же данные, но для Маши: номер q и расстояние m

Все числа целые и не превосходят 10<sup>8</sup>

### **Формат вывода**

В ответе укажите только одно число - максимальное количество покрашенных деревьев 

### Пример
```
0 3     ->     
10 2    ->    10

0 3     ->
6 10    ->    26
```

Заметим, что во 2 и 3 пунктах максимальная из левых границ отрезков лежит левее минимальной из правых границ отрезков. Таким образом мы сможем написать условие, чтобы отделить 1 и 2, 3 пункты. Данное решение работает за O(1)

```python
p, v = map(int, input().split())
q, m = map(int, input().split())
left_v, right_v = p - v, p + v #границы отрезка Васи
left_m, right_m = q - m, q + m #границы отрезка Маши
if max(left_v, left_m) <= min(right_v, right_m): #2 и 3 случаи
    print(max(right_v, right_m) - min(left_v, left_m) + 1)
else:
    print((right_v - left_v + 1) + (right_m - left_m + 1))
```

# Задача 3 (Наибольшее произведение)

В списке с произвольными числами требуется найти два таких, что их произведение будет максимальным. Гарантируется, что ответ единственный

### **Формат ввода**

В первой строке передаются числа из списка

### **Формат вывода**

Выведите два числа, дающих наибольшее произведение в порядке неубывания

### Пример
```
1 2 3 4 5    ->    4 5
-4 2 -5 3    ->    -5 -4
```

## Объяснение решения

Два числа дают наибольшее произведение, если:
1) Они оба положительные
2) Они оба отрицательные
   
Для решения задачи нам требуется найти два максимальных числа в списке и два минимальных. Тут можно привести два решения

### Решение за O(NlogN)
Данное решение использует стандартную сортировку языка, работающую за O(NlogN). Благодаря ей легко вычисляются максимальные и минимальные элементы
```python
arr = list(map(int, input().split()))
arr.sort()

candidate1 = arr[0] * arr[1]
candidate2 = arr[-1] * arr[-2]

if candidate1 > candidate2:
    print(arr[0], arr[1])
else:
    print(arr[-2], arr[-1])
```

### Решение за O(n)
В этом решении мы за один проход по массиву ищем эти элементы
```python
arr = list(map(int, input().split()))
mx_1 = max(arr[0], arr[1])
mx_2 = min(arr[0], arr[1])
mn_1 = min(arr[0], arr[1])
mn_2 = max(arr[0], arr[1])

for i in range(2, len(arr)):
    if (arr[i] >= mx_1):
        mx_2 = mx_1
        mx_1 = arr[i]
    elif (arr[i] >= mx_2):
        mx_2 = arr[i]

    if (arr[i] <= mn_1):
        mn_2 = mn_1
        mn_1 = arr[i]
    elif (arr[i] <= mn_2):
        mn_2 = arr[i]

if (mx_1 * mx_2 > mn_1 * mn_2):
    print(mx_2, mx_1)
else:
    print(mn_1, mn_2)
```

# Задача 4 (дома и магазины)

Вдоль дороги установлены 10 зданий. Среди них есть обычные жилые здания, магазины и офисы. Некоторые жилые здания достаточно далеко расположены от магазинов. Нужно найти какое максимальное расстояние требуется пройти жильцам дома до ближайшего магазина

### **Формат ввода**

В первой строке передается список зданий. 0 - офисные здания, 1 - жилой дом, 2 - магазин. Гарантируется, что на улице есть хотя бы 1 жилой дом и хотя бы 1 магазин

### **Формат вывода**

В ответе укажите число - максимальное расстояние до ближайшего магазина среди всех домов

*Примечание: если два дома - соседние, то расстояние между ними равно 1; если между ними есть 1 здание, то расстояние равно 2, и т.д*

### Пример
```
0 1 2 1 0 1 1 0 0 2    ->    3
```

## Объяснение решения
Наивное и простое решение полным перебором будет работать за O(n<sup>2</sup>). Можно добиться решения за O(n)

Пройдемся по массиву домов 2 раза. Сначала для каждого жилого дома определим ближайший левый магазин, а потом ближайший правый. Отсюда сможем найти максимальное расстояние до ближайшего магазина

```python
buildings = list(map(int, input().split()))
left = [0] * len(buildings)
shop_pos = -100
for i in range(len(buildings)):
    if buildings[i] == 2: #идем слева направо и сохраняем позицию последнего левого магазина
        shop_pos = i
    elif buildings[i] == 1: #для каждого жилого дома в массив left будем записывать расстояние до ближайшего левого магазина
        left[i] = i - shop_pos

ans = 0
shop_pos = 100
for i in range(len(buildings) - 1, -1, -1):
    if buildings[i] == 2: #идем справа налево и запоминаем позицию последнего правого магазина
        shop_pos = i
    elif buildings[i] == 1:
        min_dist = min(left[i], i - shop_pos) #запоминаем расстояние до ближайшего магазина
        ans = max(ans, min_dist)
print(ans)
```
