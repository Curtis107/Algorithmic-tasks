# Задача 1 (различные чилса)

На вход подается список чисел. Нужно определить количество различных значений

### **Формат ввода**

В первой строке передается список чисел (не больше 10<sup>5</sup> элементов)

### **Формат вывода**

В ответе укажите количество различных чисел

### Пример
```
1 2 3 4 5  ->  5
3 2 3 8 2  ->  3
```

## Объяснение решения

Для решения данной задачи подойдет такая структура данных как множество: в нем каждый элемент может храниться только 1 раз. В питоне оно задается как *set()*. Добавление элемента *x* происходит через метод *add()*: *add(x)*

```python
s = set()
arr = list(map(int, input().split()))
for i in arr:
    s.add(i)
print(len(s))
```

# Задача 2 (было ли раньше?)

Во входной строке передан список элементов. Для каждого числа нужно сказать, встречалось ли оно раньше или нет, после чего вывести ответ на экран

### **Формат ввода**

На вход подается список чисел

### **Формат вывода**

В ответе для каждого числа напечатайте 'YES', если оно уже встречалось, и 'NO' в противном случае

### Пример
```
1 2 3  ->  NO
           NO
           NO

4 5 4  ->  NO
           NO
           YES
```

## Объяснение решения

В этой задаче можно так же воспользоваться множетсвом, но попробуем реализовать решение, используя словарь. Словарь - структура данных, в которой элементы хранятся по системе ключ-значение. В питоне оно задается как *dict()* или *{}*. В этом решении ключом у нас будет выступать само число, а значением единица. Добавление нового ключа *x* со значением *y* происходит через метод *update()*: *update({x: y})*

```python
d = dict()
arr = list(map(int, input().split()))
for i in arr:
    if i not in d:
       print('NO')
       d.update({i: 1})
    else:
       print('YES')
```

s1 = set(map(int ,input().split()))

# Задача 4 (школьники и языки)

Всего *n* учеников. Каждый *i*-й ученик знает *M<sub>i</sub>* языков. Опеределите языки, которые знают все школьники, и языки, которые знает хотя бы один школьник

### **Формат ввода**

В первой стороке передается число *n* - количество учеников (1 <= *n* <= 1000)

Далее идёт *n* блоков - информации об учениках. В каждом *i*-м блоке в первой строке передается число *M<sub>i</sub>* - количество языков, который знает ученик *i* (1 <= *M<sub>i</sub> <= 500). Далее в последующих *M<sub>i</sub>* строках записаны языки - по одному в строке

### **Формат вывода**

В первой строке укажите количество языков, которые знают все ученики. Со второй строки - перечисление этих языков (каждыый на новой строке)

Потом укажите количество языков, которые знает хотя бы один ученик. Со следующей строки - перечисление этих языков (каждый на новой строке)

### Пример
```
3
3            
Russian            1
English            Russian
Chinese            3
2            ->    Russian   
Russian            Chinese
English            English
1
Russian
```

## Объяснение решения

Воспользуемся словарями и множествами (но вообще, можно было и без множеств). Ключами в словарях будут названия языков, значениями - количество учеников, что их знает. В множестве будем просто хранить языки, которые знают все. В решении будет использован метод *keys()* - он возращает список всех ключей в словаре

```python
n = int(input())
all_lang = dict() #словарь с языками и количеством учеников, знающих их
s = set()
for i in range(n):
    mi = int(input())
    for j in range(mi):
        lang = input()
        if lang not in all_lang: #добавляем новый язык в словарь и ставим ему значени 0
            all_lang.update({lang: 0})
        all_lang[lang] += 1 #увеличиваем значение на 1

for l in list(all_lang.keys()):
    if all_lang[l] == n: #если язык знают все - добавляем в множество
        s.add(l)
            
print(len(s))
print('\n'.join(s))
print(len(all_lang.keys()))
print('\n'.join(all_lang.keys()))
```

# Задача 5 (гонка черепах)

В гонке участвует *n* черепах. Каждая из черепах говорит сколько черепах перед ней и позади нее, но они могут врать. Нужно определить какое максимальное количество черепах может говорить правду

### **Формат ввода**

В первой строке передается число *n* - количество черепах в гонке (1 <= *n* <= 10000)

В следущих *n* строках записаны показания каждой из черепах: число *a<sub>i</sub>* черепах перед ней и число *b<sub>i</sub>* позади нее (|*a<sub>i</sub>*|, |*b<sub>i</sub>*| <= 10000)

### **Формат вывода**

В ответе укажите максимальное число черепах, которые могут говорить правду

### Пример
```
3
2 0
0 2    ->    2
2 2

5
0 4
1 3
2 2    ->    5
3 1
4 0
```

## Объяснение решения

В каком случае черепаха может врать? Если суммарное количество черепах перед ней и позади нее не равно *n*-1, либо если две черепахи находятся на одном и том же месте, либо число черепах спереди или сзади отрицательное (обратите внимание, что значения ограничены по модулю). Если первое и последнее условие мы легко сможем обработать, то для второго нам помогут множества

```python
n = int(input())
s = set()
for i in range(n):
    a, b = map(int, input().split())
    if (a + b) == (n - 1) and (a >= 0) and (b >= 0):
        s.add((a, b))
print(len(s))
```

Вам дан список из *n* пар слов. Каждая пара слов являются синонимами. Все слова различны. Для данного слова выведите его синоним

### **Формат ввода**

В ответе укажите слово, которое является синонимом к данному

### Пример
```
3
Good Brilliant
Wow OMG        ->        Hello
Hello Hi
Hi
```

## Объяснение решения

Для решения этой задачи идеально подходит словарь. Ключом будет один синоним, а значением другой. Будем добавлять пару дважды, меняя синонимы местами (ключ и значение)

```python
d = dict()
n = int(input())
for _ in range(n):

```

Вам даны данные о продажах некоторых товаров некоторым покупателям в разном количестве. Выведите в лексикографическом порядке фамилии покупателей, товары которые они покупали и количество товаров данного типа за все время

### **Формат ввода**

Данные передаются через файл. В каждой строке записаны данные о некоторой покупке в формате *фамилия товар количество*

### **Формат вывода**

В ответе выведите все в лексикографическом порядке. Сначала напечатайте фамилию покупателя, двоеточие. На новой строке название товара (тоже в лексикографическом порядке) и через пробел количество купленного товара за все время. На следующей строке другой товар и его количество (если товары еще есть)

### Пример
```
Ivanov paper 10            Ivanov:
Ivanov pen 4               paper 10 
Petrov cup 2               pen 5 
Sidorov paper 3        ->  Petrov:  
Ivanov pen 1               cup 2 
Petrov scissors 2          scissors 2
                           Sidorov:

## Объяснение решения

Решение данной задачи довольно простое - нужно сделать словарь в словаре. Чтобы запомнить покупателей, заведем словарь, где ключом будет фамилия покупателя, а значением сделаем словарь, в котором ключом будет название товара, а значением его количество. ```python
d = {}
file = open("input.txt", 'r')
for line in file:
    line = line.strip()
    customer, goods, amount = line.split()
    if customer in d:
        if goods in d[customer]:
            d[customer][goods] += int(amount)
        else:
            d[customer].update({goods: int(amount)})
    else:
        d[customer] = dict()
        d[customer][goods] = int(amount)

for customer in sorted(d):
    print(customer + ':')
    for goods in sorted(d[customer]):
        print(goods, d[customer][goods])
```

# Задача 8 (банковская система)

Существует банк, в котором пользователям доступны следующие действия: пополнение счета, снятие денег со счета, запрос об остатке на счете, перевод денег на счет другого пользователя, начисление процентов всем клиентам. Необходимо реализовать такую систему и для каждого запроса на остаток, банк должен выводить количество денег на счету у пользователя (положительное, отрицательное, либо 'ERROR', если у клиента не открыт счет в этом банке)

### **Формат ввода**

Данные передаются в файле. В каждой строке записана какая-то из 5-ти операций. Возможны следующие операции: 'DEPOSIT' *name* *sum* - пополнение счёта клиента с именем *name* на сумму *sum* (если счета нет - он создается); 'WITHDRAW' *name* *sum* - снять со счета клиента с именем *name* сумму *sum* (если счета нет - он создается); 'BALANCE' *name* - запрос на остаток средств на счету у клиента с именем *name*; 'TRANSFER' *name1* *name2* *sum* - перевод средств со счета клиента с именем *name1* клиенту с именем *name2* в размере *sum* (если у кого-то из них нет счета - он создается); 'INCOME' *p* - начислить всем клиентам, у кого открыт счет в банке, *p*% от суммы на счете (проценты начисляются только тем, у кого положительный баланс; дробная часть начисленных процентов отбрасывается)

### **Формат вывода**

Для каждого запроса об остатке средств выведите количество денег на балансе, если счет не открыт - выведите 'ERROR'

### Пример
```
DEPOSIT Ivanov 100
INCOME 5
BALANCE Ivanov                    105
TRANSFER Ivanov Petrov 50    ->   -50 
WITHDRAW Petrov 100               ERROR 
BALANCE Petrov
BALANCE Sidorov
```

## Объяснение решения

Порядок хранения данных схож с задачей №7: ключом будет фамилия, а значением баланс пользователя

```python
bank = {}
file = open("input.txt", 'r')
for line in file:
    line = line.strip()
    arr = list(line.split())
    if arr[0] == 'DEPOSIT':
        name, summ = arr[1], int(arr[2])
        if name not in bank:
            bank.update({name: 0})
        bank[name] += summ

    elif arr[0] == 'WITHDRAW':
        name, summ = arr[1], int(arr[2])
        if name not in bank:
            bank.update({name: 0})
        bank[name] -= summ

    elif arr[0] == 'BALANCE':
        name = arr[1]
        if name not in bank:
            print('ERROR')
        else:
            print(bank[name])

    elif arr[0] == 'TRANSFER':
        name1, name2, summ = arr[1], arr[2], int(arr[3])
        if name1 not in bank:
            bank.update({name1: 0})
        if name2 not in bank:
            bank.update({name2: 0})
        bank[name1] -= summ
        bank[name2] += summ

    elif arr[0] == 'INCOME':
        p = int(arr[1])
        for names in bank:
            if bank[names] > 0:
                bank[names] += int(bank[names] * (p / 100))
```

# Задача 9 (палиндром)

Дана строка с заглавными латинскими буквами. Из данных букв нужно составить палиндром наибольшей длины (буквы можно переставлять местами и удалять). Если палиндромов несколько, выведите тот, что первый в алфавитном порядке

### **Формат ввода**

В первой строке передается число *n* - количество букв

Во второй строке передается строка из *n* букв

### **Формат вывода**

Выведите самый длинный палиндром

### Пример
```
ABB    ->    ABA
ABBCC    ->    BCACB
```

## Объяснение решения

Будем сразу строить палиндром в лексикографическом порядке. Заведем словарь, в котором ключом будет буква, а значением - количество раз, которое она встречалась

Палиндром - симметричная последовательность букв относительно своего центра. То есть, чтобы написать целое слово-палиндром, нам достаточно значть половину слова, может быть включая центральный элемент. Допустим, что буквы 'A' и 'B' встречались по 2 раза, тогда в ответе будет 'ABBA'. Но что если 'A' - 3 раза, 'B' - 4 раза, 'C' - 1 раз? Так как палиндром симметричный, то мы можем использовать только четное количество букв, значит мы используем две буквы 'A' и четыре буквы 'B' (получается 'ABBBBA'). У нас отсается одна буква 'A' и одна буква 'C'. Заметим, что слово может быть палиндромом, если центральная буква встречается нечетное количество раз (т.е в нашем случае мы можем в центр поставить 'A' или 'C': 'ABBABBA' и 'ABBCBBA', но так как нужен первый в алфавите, то ответом будет первый вариант). Но если бы букв 'C' было бы 3, то ответ был бы такой: 'ABBCCCBBA'

*Примечание: в коде используется метод get() с параметрами i и 0. Он возвращает значение по ключу i, но если такого ключа нет, то 0*

```python
n = int(input())
d = {}
str = input()
for i in str:
    d[i] = d.get(i, 0) + 1

ans = []
letter = ''
for key in sorted(d):
    if d[key] > 1:
        ans.append(key * (d[key] // 2))
        if (letter == '' or key < letter) and d[key] % 2 != 0:
            letter = key
    elif d[key] == 1 and letter == '':
        letter = key
print(''.join(ans) + ''.join(letter) + ''.join(ans[::-1]))
```

# Задача 10 (расшифровка письменности)

Дана последовательность значков S и набор из g значков, составляющих некоторое неизвестное слово W. Нужно найти количество всех непрерывных фрагментов в последовательности S длины g, которые являются какой-либо перестановкой значков из W (порядок значков может быть любым, так как оригинальный порядок слова неизвестен из-за эстетических особенностей древней письменности).

### **Формат ввода**

В первой строке передаются числа *g* и |*S*|; *g* – количество значков в слове *W* 1 ≤ *g* ≤ 3 000, *g* ≤ |*S*| ≤ 3 000 000 где |*S*| – количество значков в последовательности *S*

Во второй строке передается *g* последовательных символов

В третьей строке передается последовательность *S*

### **Формат вывода**

В ответе укажите число возможных вхождений слова *W* в последовательность *S*

### Пример
```
4 11
cAda            ->    2
AbrAcadAbRa
```

## Объяснение решения

Так как слово мы не знаем, то оно может состоять из любой последовательности этих значков. В примере слово длины 4, а это 4! = 24 варианта, для слова длины 6 - 720 вариантов, для слова длины 10 - 3 628 800 вариантов. Перебором действовать не выход

Для начала заведем словарь, в котором ключом будет значок слова *W*, а значением количество его вхождений в это слово. Теперь нам нужно придумать способ, как нам подсчитать эти слова. Довольно очевидная вещь, что в слове должны использоваться все значки. Давайте будем действовать способом фиксированного окна. Представьте, что окно - это объектив камеры, а буквы в последовательности *S* - шеренга людей (одинаковых по комплекции). Если фотограф будет двигаться вдоль шеренги, то из кадра одновременно будет "выпадать" один человек и "приходить" новый. Здесь то же самое: мы будем двигать это окно, и на каждой итерации будет выскакивать одна буква и приходить новая. Если мы знаем, какие буквы внутри этого окна, то можем сопоставить их со словарем и понять, используются ли все буквы или нет. Если нет - то это не может быть словом *W*, а если все, то может. При таком подходе очень удобно следить за количеством букв, потому что если уходит одна буква, мы либо увеличиваем количество недостающих букв, либо нет (в зависимости от того, есть ли она в слове *W*), а с новой приходящей буквой все так же

```python
g, s = map(int, input().split())
str1 = input()
d = {}
for i in str1:
    d[i] = d.get(i, 0) + 1
str2 = input()

unused = g
count = 0
start = 0
last = g - 1
for i in range(start, last + 1):
    if str2[i] in d:
        d[str2[i]] -= 1
        if d[str2[i]] >= 0:
            unused -= 1

if unused == 0:
    count += 1

start += 1
last += 1
while last < s:
    if str2[start - 1] in d:
        if d[str2[start - 1]] >= 0:
            unused += 1
        d[str2[start - 1]] += 1

    if str2[last] in d:
        if d[str2[last]] > 0:
            unused -= 1
        d[str2[last]] -= 1

    if unused == 0:
        count += 1

    start += 1
    last += 1

print(count)
```

# Задача 11 (игра в баскетбол)

*n* человек играют в настольный баскетбол. Игрок может забить мяч в любое из двух колец и получить за это от 1 до 3 очков. Определите человека, кто получил больше всего очков и количество этих очков

### **Формат ввода**

В первой строке передается число *n* - количество игроков (1 <= *n* <= 50)

В следующих *n* строках передается по одному имени игрока *s<sub>i</sub>* (1 <= |*s<sub>i</sub>*| <= 15)

В *n*+2 строке передается число *m* - количество записей счета (1 <= *m* <= 500)

В следующих *m* строках передаются записи счета в формате *a*:*b* *name*. *a* - количество очков, забитых в первое кольцо, *b* - количество очков, забитых во второе кольцо, *name* - имя игрока, кто изменил счет. Гарантируется, что все записи корректны

### **Формат вывода**

В ответе укажите через пробел имя игрока, набравшего максимальное количество очков, и количество очков

### Пример
```
3
Vasya
Petya
Masha
4            ->    Masha 4
3:0 Vasya
3:2 Petya
4:2 Masha
7:2 Masha
```

## Объяснение решения

Создадим словарь, ключом сделаем имя игрока, а значением количество забитых очков

```python
n = int(input())
names = {}
for i in range(n):
    names.update({input(): 0})

m = int(input())
left, right = 0, 0
for i in range(m):
    ai, str = input().split(':')
    bi, name = str.split()
    ai, bi = int(ai), int(bi)
    if left != ai:
        names[name] += (ai - left)
        left = ai
    if right != bi:
        names[name] += (bi - right)
        right = bi

mx = -1
name = ''
for i in names:
    if names[i] > mx:
        mx = names[i]
        name = i
print(name, mx)
```
